# Personal Trainer App

This is a React-based application created for Front end -ohjelmointi - SOF004AS3A-3028 course in Haaga Helia. It can be used to manage customers and trainings. It also has a calendar. It uses MaterialUI for tables and FullCalendar for calendar.

## Features
- **Customer Management**: Add, edit, and delete customers.
- **Training Management**: Manage training sessions (linked to customers)
- **Calendar View**: Visualize training sessions in a calendar.

## Tech Stack
- React, TypeScript, Vite
- MaterialUI, FullCalendar
- REST API (https://juhahinkula.github.io/personaltrainerdocs/)
